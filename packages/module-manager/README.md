## Serenity/JS Module Manager

[`@serenity-js/module-manager`](https://serenity-js.org/api/module-manager)
helps you keep your Serenity/JS project up to date with dependency updates.

### Installation

To install this module, run the following command in your computer terminal:

```sh
npm install --save-dev @serenity-js/module-manager
```

To learn more about Serenity/JS and how to use it on your project, follow the [Serenity/JS Getting Started guide](https://serenity-js.org/handbook/getting-started/).
